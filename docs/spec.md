# 小红书图片生成器（纯前端 Konva 版）

## 目标（MVP）
- 纯前端：React + Vite + TS
- 画布：1080x1440
- 模板：先实现 1 个 pinkStack（贴近截图）
- 编辑：
    - 文字：可新增、拖拽、双击编辑（textarea 覆盖方式，支持换行）
    - 图片：本地上传插入、拖拽、缩放
- 多页：上一张/下一张（每页独立保存节点状态）
- 导出：下载 PNG（pixelRatio=2，导出时隐藏选中框）

## UI
- 顶部：Toolbar（模板、上一张、下一张、添加文字、插图、下载）
- 中间：画布区域（居中显示，可用 CSS 缩放适配屏幕）
- 右侧：属性面板（选中文本时：字号/颜色/行高/对齐；选中图片时可选：旋转）

## 关键实现要求
1) 必须使用 react-konva
2) 双击文本编辑：创建 textarea 覆盖在对应位置，blur 后写回 Konva Text，支持换行
3) 选中框 Transformer：选中元素才显示；导出时隐藏，导出后恢复
4) 多页状态：pages[pageIndex].nodes 存储当前页 nodes，切页前保存，切页后加载
5) 导出：stage.toDataURL({ pixelRatio: 2 })

## 模板 pinkStack
- 背景：粉色竖向渐变
- 三层叠卡（两层背板 + 一层主白卡）
- 三段文本（标题/强调/说明）
- 中部信息卡：白色圆角矩形 + 阴影 + 内部可替换图片占位
- 底部头像圆形 + 昵称
